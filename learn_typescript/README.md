# Learn TypeScript Through This Repo

This folder stores TypeScript learning notes based on the FreeCAD repository, especially the AsterForge frontend in `variants/asterforge/frontend/app`.

Structure:

- `session-001.md`: running log of this conversation's learning content
- `lesson-001-types-from-real-source.md`: first lesson using real project files
- `lesson-002-protocol-contracts.md`: how the frontend API contract is typed
- `lesson-003-react-state-typing.md`: how React state is typed in the main app shell
- `exercises-001.md`: practice tasks based on the same source files
- `lesson-004-step-client-typing.md`: typed API helpers and render-prep data transforms
- `lesson-005-utility-function-typing.md`: typed utility functions and domain logic
- `exercises-001-answer-key.md`: worked answers for the first exercise set
- `lesson-006-typed-test-patterns.md`: how typed mocks and typed module imports work in Vitest
- `lesson-007-generated-contract-flow.md`: how generated interfaces flow into the app
- `lesson-008-protocol-refinements.md`: how protocol aliases refine generated types with `Omit`
- `lesson-009-component-prop-typing.md`: how React component props are typed in `App.tsx`
- `session-002.md`: exercise-driven continuation prompts for the next study pass
- `runnable-examples-001.md`: executable TypeScript example chunks tied to a Vitest file
