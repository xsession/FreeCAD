# Qt-to-TypeScript Tracker-Ready Issues

Status: local tracker-ready issue files

## Purpose

These files convert the milestone issue pack into one-file-per-issue artifacts that can be copied into a real tracker later.

Remote issue creation is not available through the current tool set, so this directory is the in-repo substitute.

## File Format

Each issue file is structured as:

- outcome
- why this matters
- primary scope
- in scope and out of scope
- deliverables
- repo anchors
- dependencies
- acceptance checklist
- risks and notes
