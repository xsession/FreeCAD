# Points Module

> **Source**: `src/Mod/Points/` · ~15 .cpp · ~14 .h · ~4 .py  
> **Dependencies**: FreeCADApp, FreeCADBase

## 📋 Overview
Point cloud import/export and visualization. Supports E57, PLY, PCD, ASC formats. Key classes: `PointsFeature`, `PointKernel`, `PropertyPointKernel`.

## Key Features
- Import point clouds from laser scanners / photogrammetry
- Visualize colored point clouds in 3D viewer
- Convert to mesh via surface reconstruction
- Formats: E57, PLY, PCD, ASC, XYZ
