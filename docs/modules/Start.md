# Start Module

> **Source**: `src/Mod/Start/` · ~18 .cpp · ~19 .h · ~3 .py  
> **Dependencies**: FreeCADGui

## 📋 Overview
The Start page displayed when FreeCAD launches. Shows recent files, examples, and getting-started resources.

## Key Features
| Class | Purpose |
|---|---|
| `StartView` | Custom MDI view for the start page |
| `RecentFilesModel` | Recent file list with thumbnails |
| `ExamplesModel` | Built-in example file browser |
| `FileCardDelegate` | Card-style file display widget |
